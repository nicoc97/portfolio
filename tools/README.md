# Development Tools

This folder contains build and development tools:

- `lighthouse-audit.cjs` - Lighthouse performance auditing
- `prerender.js` - Static site prerendering
- `simple-lighthouse.cjs` - Simplified lighthouse runner

## Usage

Run lighthouse audits:
```bash
npm run lighthouse
npm run lighthouse:save
npm run lighthouse:ci
```

Build with prerendering:
```bash
npm run build:ai-optimized
```
